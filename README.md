# Projeto RHSystem (SQLite)

Repositório com scripts SQL do mini-mundo RH, pronto para uso no **SQLite / SQLiteStudio**.

## Estrutura
- `sql/create_tables.sql` — criação das tabelas (SQLite)
- `sql/inserts.sql` — comandos INSERT para povoar o banco
- `sql/selects.sql` — exemplos de consultas (SELECT)
- `sql/updates_deletes.sql` — exemplos de UPDATE e DELETE

## Como usar (SQLiteStudio)
1. Abra o **SQLiteStudio**.
2. Crie um novo banco de dados.
3. Abra o arquivo `sql/create_tables.sql` e execute (ou copie/cole no SQL editor).
4. Execute `sql/inserts.sql` para popular as tabelas.
5. Rode as consultas em `sql/selects.sql` e os scripts de modificação em `sql/updates_deletes.sql`.
